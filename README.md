## DMG Mori NTX Specialization

I have hands-on programming experience with the DMG Mori NTX series (NTX 2500, NTX 3000) — multi-tasking turn/mill centers running Fanuc 31i-B under the CELOS interface.

Key areas I work in:
- Sub-spindle synchronization — G114/G115 spindle sync, phase alignment, M-code handshake sequencing
- B-axis milling — compound angle toolpaths, tilted plane machining (G68.2), tool orientation
- Simultaneous turn/mill — overlapping spindle and live tool operations for cycle time reduction
- Work shift & offset management — G54–G59 extended offsets, WORK SHIFT register, sub-spindle Z-offset calibration
- CELOS app integration — Job Manager, Tool Management, Machine Protection
- Fanuc macro programming — custom G-code macros (#variables, DPRNT, tool life logic)

Currently documenting NTX programming patterns and G-code templates. If you work on DMG Mori NTX machines, feel free to open a discussion.
